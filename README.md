# NICA Academy · Web & AI Integrations

Personal landing page and demo of AI integrations by **Aurel Nica**  
Stack: HTML/CSS/JS · Python/Flask · OpenAI API.

## Live

- Frontend (GitHub Pages): `https://nica71.github.io/nica-portfolio/`
- Backend (Render/Railway): `https://<app>.onrender.com`

## Features

- 🔹 Multi-language landing (RO/RU/EN)
- 🔹 Sticky header, scroll reveal animations
- 🤖 AI Chat widget (OpenAI gpt-4o-mini)
- 🌦 `/api/weather` — current weather via Open-Meteo
- 📝 `/api/summarize` — text summarization via OpenAI
